# TTY Fireworks :fireworks:
> Fireworks for your terminal


## Installation
```bash
  gem install tty-fireworks
````

## Usage
```bash
  fireworks
```

 - `q` quits
 - `c` clears the screen
 - all other keys fire a salvo
